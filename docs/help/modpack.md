# 导出整合包

::: info 该文档编写者
- Capslock800000 - 2026/3/27
:::


## 前言

此文档将会教你如何导出整合包。
小白友好向。

## 开始

启动器提供了导出整合包页面,你可以通过主菜单进入导出整合包页面:

![步骤1](/zh/docs/modpack/step1.png)

![步骤2](/zh/docs/modpack/step2.png)

### 选择要导出的整合包类型

![选择导出类型](/zh/docs/modpack/select.png)

类型:
- MCBBS整合包格式:多数国内启动器支持
- Modrinth整合包格式:绝大多数启动器支持
- CurseForge整合包格式:绝大多数启动器支持
- MultiMC整合包格式:绝大多数启动器支持

### 准备导出

以Modrinth整合包格式为例。
选择格式后来到这个界面:

![信息填写](/zh/docs/modpack/export1.png)

填写完毕后即可来到这个界面:

![选择要打包的文件](/zh/docs/modpack/export2.png)

在这个界面选择要打包的资源文件,选择要保存的整合包文件的目录,即可开始导出。

![导出](/zh/docs/modpack/export3.png)

完成:

![导出完毕](/zh/docs/modpack/export4.png)
导出完毕后即可在你选择的目录里找到你生成的整合包文件。
